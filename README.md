# clear_dataframe

Description. 
The clear_dataframe is used to:
	- remover dados duplicados
	- remover dados nulos

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install clear_dataframe

```bash
pip install clear_dataframe
```
